Программа для розыгрыша игрушек

Порядок работы:
1. Игрушки добавляются в динамический массив toys
2. При необходимости можно вывести весь список игрушек (меню - 1)
3. Есть возможность добавления новой игрушки в динамический массив toys (меню - 2)
4. Розыгрыш игрушек:
    4.1 Сперва заполняется динамический массив firstDischargeArray id игрушек в соответствии 
        с их весом (вероятностью выпадения)
    4.2 Поскольку у каждой игрушки есть значение весомости (вероятности выпадения), то 
        id игрушек перезаполняются в динамический массив raffleArray, путем перемешивания
        id игрушек небольшим алгоритмом
    4.3 В результате происходит выбор псевдослучайного id игрушки, вывод сообщения на экран 
        о выигрыше и удаления игрушки из динамического массива toys.
5. Программа записывает результаты розыгрыша в файл Notes.txt
6. При повторном запуске программы файл очищается от прошлых результатов розыгрыша
